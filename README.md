# Website

Welcome to this project repository.

## Getting Started

To get started with this project, clone the repository and navigate to the project directory.

```bash
git clone <repository-url>
cd website
```

## Installation

Install dependencies:

```bash
npm install
```

## Usage

Run the development server:

```bash
npm run dev
```
